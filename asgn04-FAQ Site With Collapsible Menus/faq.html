Frequently Asked Questions

What is this project about?

This project demonstrates an accessible FAQ accordion using semantic HTML, minimal JavaScript, and smooth CSS animations.

Why use the <details> element?

The <details> element provides built-in keyboard accessibility and screen reader support without extra ARIA scripting.

Is JavaScript required?

JavaScript enhances the experience with smooth animation and controlled toggle behavior, but the core functionality works without it.

How is accessibility handled?

Native semantic elements, proper focus handling, and reduced-motion support ensure the component remains accessible.

AI Usage Note

AI was used to propose accessible accordion patterns and draft concise FAQ microcopy. The final implementation was reviewed and adjusted to prioritize semantic HTML, minimal JavaScript, and clean DOM updates. Animation and styling choices were refined manually.
